
$Y**Arrays**$W

Arrays works inside strings by using
square brackets to differentiate items.

Item can have value, and a key.

Example of array item containing only value
  ["im value"]

Example of array item containing value and key
  ["im key" : "im value"]

Example of array
  {[123][234]["stuff":"block"]}

Example of array containing two arrays
  {
	  [{[123][234][345]}]
	  [{[456][567][678]}]
  }
